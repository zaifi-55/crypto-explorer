# Crypto Price Explorer & Predictor
An end-to-end ML pipeline built during my August 2025 internship.

### Key Features
* **Data Ingestion:** CoinGecko API integration with local caching.
* **Feature Engineering:** SMA, RSI, Momentum, and Lagged Returns.
* **Modeling:** Time-series forecasting using RandomForest, Ridge, and XGBoost.
* **UX:** Interactive Streamlit dashboard for backtesting and predictions.

### Tech Stack
Python, Streamlit, Pandas, Scikit-Learn, Plotly.
